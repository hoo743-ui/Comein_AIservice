# 04. GUI · UX

## 무드

**정제된 "assistant(비서)" 프리미엄** — 차분한 뉴트럴 표면 + 연보라/딥 인디고 액센트(절제). 전형적 AI-생성 클리셰(연보라 범벅·과한 글래스·큰 라운드)를 피하고, 크리스프 헤어라인·뉴트럴 잉크 섀도우·에디토리얼 세리프로 격을 준다. (근거: ADR-003)

## 3분할 레이아웃 (워크스페이스)

```
┌────────┬─────────────────────────┬──────────────┐
│ 사이드바 │      중앙(Chat/기능)       │  컨텍스트 패널  │
│ 새 대화 │  대화 스트림 + 인라인 카드   │ 다가오는 일정   │
│ 검색   │  (일정/메모/Todo 카드)     │ 임박한 Todo   │
│ 메뉴   │                         │ 최근 메모      │
│ 대화목록│  [입력창 ────────── ➤]   │ (채팅 홈에만)  │
└────────┴─────────────────────────┴──────────────┘
```

- **Chat이 홈** — "대화가 곧 인터페이스".
- AI 생성 결과는 **인라인 카드**로 렌더 + 확인/수정/취소(일정은 pending→confirm).
- 컨텍스트 패널은 **채팅 홈에서만** 표시(기능 페이지는 넓게).

## 입체(depth) 시스템

- `.elevated` — 카드가 떠 있는 느낌(상단 inset 하이라이트 + 레이어드 섀도우 + 카드색 그라디언트).
- `.orb-3d` — 광택 있는 구체(채팅 오브 등).
- `bg-app` — 중앙 브랜드 글로우 + 코너 라이팅 + 비네트로 캔버스 깊이.
- 컬럼/트레이는 `bg-muted/40`로 recessed → 카드가 상대적으로 부각.

## 모션

- **입장 리추얼**: 입장하기 → 첫 페이지가 투명해지며 **Comein 마크 스플래시** 등장 → 워크스페이스로 이어짐(크로스 디졸브). 로고 배경은 딥 인디고 라디얼로 이음새 제거. (ADR-005)
- **바람 앰비언트**: 랜딩·로그인 배경 사진에 켄번즈 드리프트 + 빛 스윕(`animate-kenburns`/`animate-sheen`, tailwind config 키프레임). 브랜드 모먼트라 항상 재생.
- 마이크로: 카드 hover 리프트, 버튼 press.

## 타이포그래피

- 디스플레이/헤드라인: **Fraunces**(세리프).
- 한글/본문: **Pretendard**. 라틴 본문 Inter.
- 숫자+한글 혼합 라벨(예: "2026년 7월")엔 세리프 금지(글리프 폴백 불일치) → 본문 폰트 + `tabular-nums`.

## 히스토리(대화) UX — 차별화

"채팅이 쌓이면 못 찾는다"를 정면으로:
- **핀 고정**: 중요한 대화를 상단 "고정" 섹션에.
- **생성물 카운트 배지**: 각 대화가 만든 항목 수 → *결과물로 찾기*.
- **검색** + 활성/호버 정돈. (향후: 결과물↔대화 양방향 앵커, 유형 패싯.)

## 접근성

- 색 대비는 토큰 기반으로 라이트/다크 모두 확보. 포커스 링(`focus:ring-ring`).
- 투명(글래스) 요소는 완전 투명 대신 **반투명 + backdrop-blur + 보더**로 가독성 유지.
